Meet App

A serverless, progressive web application (PWA) built with React using test-driven development (TDD). The app fetches upcoming events from the Google Calendar API, allowing users to filter events by city, view event details, and visualize data with interactive charts. Features offline functionality and can be installed as a home screen shortcut.

Technologies: React, Vite, Google Calendar API, AWS Lambda, Recharts, Jest, Puppeteer, PWA, OAuth2, Atatus
